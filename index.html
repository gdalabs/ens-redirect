ENS Redirect

Go to any ENS name's URL, instantly.

How it works

  1. Set a url text record on your ENS name at app.ens.domains
  2. Append your .eth name to this URL
  3. You get redirected to the URL you set
https://ens-redirect.pages.dev/yourname.eth

Try it

Why?

ENS names can store a url text record, but there's no easy way to visit it directly. This is a free, open-source tool to make that possible.
